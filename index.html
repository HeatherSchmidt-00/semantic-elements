What are Semantic Elements?

Semantic Elements, broken down, are elements that clearly define the content. Semantic Elements help eliminate unnecessary use of divs by using more specific descriptors of the element, which clarifies the content for the developer and the browser.


Why use Semantic Elements?

We use Semantic Elements to enhance:

  1. Accessibility
  2. Searchability
  3. Internationalization
  4. Interoperability

What are the Main Semantic Tags?

The main semantic tags consist of six elements:

  • Header
  • Nav
  • Aside
  • Section
  • Article
  • Footer

Semantic Elements are a useful tool to communicate more descriptively while clearly defining the content for developers and browsers.
